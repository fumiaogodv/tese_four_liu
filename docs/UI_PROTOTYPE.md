# 用户界面原型（CLI）

对应 **第 5 部分：用户交互的软件构造**，界面形态参考教程图 5.1 / 5.2。

## 主界面原型（图 5.1）

```
==================================================
  口算练习系统 — 统一入口 (故事1~6 集成)
==================================================
  【华经理】1~5   【小明】6
--------------------------------------------------
  [1] (华) 批量生成练习题
  [2] (华) 挑选并导出练习卷
  [3] (华) 导入答案并批改
  [4] (华) 查看练习成绩统计
  [5] (华) 错题与薄弱题目分析
  [6] (明) 交互式口算练习
--------------------------------------------------

请输入功能编号 [0]: _
```

## 交互练习子界面（小明专用）

```
--- 交互式口算练习 (小明) ---
  1. 开始新练习 (即时出题)
  2. 使用已有练习卷
请选择 [1]: 1

练习类型: 1=加法  2=减法  3=加减混合
请选择 [3]: 3
题量 (默认50，可填10便于试做) [50]: 10

开始练习，共 10 题。输入数字答案，回车确认。
提示: 输入 q 可放弃本次练习。

第  1/10 题   48+7= _
第  2/10 题   15-6= _
...

========================================
  练习完成！得分: 80.0  (8/10)
========================================

错题回顾：
  48+7 = 50  (正确答案 55)
```

## 角色与功能映射

| 编号 | 角色 | 功能 | 对应故事 |
|------|------|------|----------|
| 1 | 华经理 | 批量生成练习题 | 故事1~3 |
| 2 | 华经理 | 挑选并导出 | 故事2 打印格式 |
| 3 | 华经理 | 导入答案批改 | 故事4 数据处理 |
| 4 | 华经理 | 成绩统计 | 故事4 |
| 5 | 华经理 | 错题分析 | 故事4 |
| 6 | 小明 | 机器上交互练习 | 故事6 新增 |
| 0 | 全部 | 退出 | — |

## 设计说明

- **数字导航**：每项功能对应一个数字，减少记忆负担（教程要求）。
- **角色标注**：菜单标注「华 / 明」，区分使用者。
- **子流程返回**：子功能执行完毕自动回到主菜单，无需记忆调用顺序。
- **单一入口**：仅 `main.py` → `Application.run()` 启动。
